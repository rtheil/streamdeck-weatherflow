# WeatherFlow Tempest — Stream Deck Plugin

Displays live weather data from your [WeatherFlow Tempest](https://tempestwx.com) station on Stream Deck buttons.

> **Disclaimer.** This is an unofficial plugin and is not affiliated with, endorsed by, or sponsored by WeatherFlow Inc. "WeatherFlow" and "Tempest" are trademarks of their respective owners.

## Features

- **11 configurable metrics** per button: Temperature, Feels Like, Humidity, Dew Point, Wind Speed & Direction, Wind Gust, Barometric Pressure, UV Index, Solar Radiation, Precipitation Today, and Lightning
- **Imperial or metric units** per button
- **Configurable refresh interval** (1–30 minutes)
- Press any button to **force-refresh** immediately
- Automatic refresh of all buttons when station credentials change

## Setup

### 1. Prerequisites

- [Node.js 20+](https://nodejs.org)
- [Stream Deck software 6.6+](https://www.elgato.com/downloads)

### 2. Install & build

```bat
npm install
npm run create-icons
npm run build
```

### 3. Sideload into Stream Deck

Double-click `com.rtheil.weatherflow.sdPlugin` — Stream Deck will prompt to install it. Alternatively, copy the folder to:

- **Windows:** `%APPDATA%\Elgato\StreamDeck\Plugins\`
- **macOS:** `~/Library/Application Support/com.elgato.StreamDeck/Plugins/`

Then restart Stream Deck.

### 4. Get your WeatherFlow API token

1. Go to [tempestwx.com → Settings → Data Authorizations](https://tempestwx.com/settings/tokens)
2. Create a new Personal Access Token
3. Copy it for use in step 5

### 5. Configure

1. Drag **Weather Display** from the action list onto a button
2. In the Property Inspector (right panel):
   - Paste your **API Token**
   - Enter your **Station ID** (visible in the Tempest app or the web URL)
   - Choose what to **display** on this button
   - Pick **imperial or metric** units
   - Set the **refresh interval**

Add as many buttons as you like — each one can show a different metric.


The plugin connects to the running Stream Deck app automatically when you sideload it.

## WeatherFlow API reference

- REST base: `https://swd.weatherflow.com/swd/rest`
- Station observations: `GET /observations/station/{station_id}?token=...`
- All values are returned in metric units from the API; the plugin converts to your chosen unit system.

## Privacy

Your API token and station ID are stored locally by Stream Deck and only sent to `swd.weatherflow.com`. See [PRIVACY.md](PRIVACY.md).
